# Tarefa 1

## Descrição

Recebe uma string contendo a hierarquia de uma arvore e retorna a arvore escrita de uma outra forma.

exemplo:

input:
[A,B] [A,C] [B,G] [C,H] [E,F] [B,D] [C,E]

output:
A[[B[[G][D]]][C[[H][E[[F]]]]]]
